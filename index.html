HTML

เขียนขึ้นเมื่อวันที่ 6 สิงหาคม พ.ศ 2534

ภาษา HTML

➔ HTML เป็น ภาษามาร์กอัป (Markup Language) ใช้สำหรับกำหนดโครงสร้างของหน้าเว็บ เช่น หัวข้อ ย่อหน้า ตาราง ลิงก์ และรูปภาพ
➔ HTML ไม่ใช่ภาษาโปรแกรม เพราะไม่สามารถคำนวณ ตัดสินใจ หรือประมวลผลตรรกะได้ แต่ทำหน้าที่เป็นโครงสร้างพื้นฐานของเว็บไซต์ และมักทำงานร่วมกับ CSS และ JavaScript

CSS

เขียนขึ้นเมื่อวันที่ 17 ธันวาคม 2539

ภาษา CSS

➔ CSS เป็น ภาษากำหนดรูปแบบ (Style Sheet Language) ใช้สำหรับตกแต่งหน้าเว็บที่สร้างด้วย HTML เช่น สี ฟอนต์ ขนาด การจัดวางตำแหน่ง และการออกแบบ layout
➔ CSS ช่วยแยกเนื้อหาออกจากการออกแบบ ทำให้เว็บไซต์ดูสวยงาม เป็นระเบียบ และรองรับอุปกรณ์หลายขนาด

JAVASCRIPT

เขียนขึ้นในเดือน ธันวาคม 2538

ภาษา JAVASCRIPT

➔ JavaScript เป็น ภาษาโปรแกรมแบบสคริปต์ (Scripting Language) ใช้เพิ่มความสามารถในการโต้ตอบบนหน้าเว็บ เช่น การคลิกปุ่ม การแสดงข้อความแจ้งเตือน การตรวจสอบฟอร์ม และแอนิเมชัน
➔ JavaScript ทำงานฝั่งผู้ใช้ (Client-side) และในปัจจุบันสามารถทำงานฝั่งเซิร์ฟเวอร์ได้ด้วย เช่น Node.js

PYTHON

เขียนขึ้นเมื่อวันที่ 20 กุมภาพันธ์ 2534

ภาษา PYTHON

➔ Python เป็น ภาษาโปรแกรมระดับสูง (High-level Programming Language) ที่ออกแบบให้เขียนง่าย อ่านง่าย และเข้าใจได้รวดเร็วใช้ได้หลากหลายงาน เช่น การพัฒนาเว็บไซต์ วิเคราะห์ข้อมูล ปัญญาประดิษฐ์ (AI) เขียนโปรแกรมทั่วไป และงานอัตโนมัติ
➔ Python เป็นภาษาที่นิยมมากในปัจจุบัน ทั้งในวงการศึกษาและอุตสาหกรรม
